# README #

Terraform scripts to create an AWS VPC.

The TF plan will create the following resources:

* X1 AWS VPC
* x1 Public route table
* x1 IGW
* x2 Public subnets
* x2 Private subnets

### How do I get set up? ###

* Install Terraform
* Clone the repo
* Create terraform.tfvars
* Run terraform plan
* Run terraform apply
* Run terraform destroy
